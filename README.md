# Telegram Card Generator Bot (Demo)

⚠️ Этот проект — **универсальный генератор изображений по шаблону** (overlay текста по координатам).
Он предназначен для **легитимных** задач: открытки, сертификаты, карточки для соцсетей, статусы заказов в твоём сервисе и т.п.

## Структура
- `assets/template.png` — шаблон (946x2048)
- `fonts/` — положи сюда свои шрифты:
  - `bold.ttf` (для времени и процента)
  - `simple.ttf` (для остальных строк)
  - `mono.ttf` (для адреса)
- `main.py` — бот на aiogram 3 + Pillow

## Запуск локально
1) Установи зависимости:
```bash
pip install -r requirements.txt
```

2) Задай токен:
```bash
export BOT_TOKEN="YOUR_TOKEN"
```

3) Запусти:
```bash
python main.py
```

## Как менять координаты
Координаты лежат в `main.py` в блоке `Coords(...)`:
- `TIME_BOX`, `BATT_BOX`, `OPID_BOX`, `AMOUNT_LINE`, `WALLET_BOX`

## Примечание про шрифты
Если файлы шрифтов не найдены/не читаются, бот использует стандартный шрифт Pillow.


### Отправка без потери качества
Бот отправляет итоговый PNG как **документ**, чтобы Telegram не применял фото-сжатие.
